# REP FITNESS

REP FITNESS is a fitness website project built with TypeScript, Vite, and CSS. This project currently has a complete frontend website with multiple fitness platform sections, AI fitness concepts, app dashboard preview, workout planner ideas, trainer sections, gym logger UI, meal scan UI, and responsive layout.

## Main Files

- `src/main.ts`: starts the app.
- `src/app.ts`: assembles all website sections.
- `src/data.ts`: stores website content data.
- `src/components/`: contains separate section files.
- `src/styles.css`: contains all styling and responsive design.
- `src/interactions.ts`: contains before/after slider interaction.
- `src/media.ts`: contains SVG/image helper functions.
- `src/types.ts`: contains TypeScript types.
